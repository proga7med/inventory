# Warehouse Inventory

A simple inventory.

## Prerequisites

- JDK 11 or newer
- MariaDB

## Getting started

- Build the project using the following command

```cmd
> gradlew.bat build
```

- To run the project, run the following command

```cmd
> gradlew.bat bootRun
```

- To run tests, run the following command

```cmd
> gradlew.bat test
```
